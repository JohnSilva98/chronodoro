# Chronodoro 🍅

Um aplicativo de Pomodoro timer moderno e intuitivo desenvolvido com Next.js para ajudar você a aumentar sua produtividade através da técnica Pomodoro.

## 🎯 Sobre o Projeto

O Chronodoro é uma ferramenta de gerenciamento de tempo baseada na famosa Técnica Pomodoro, desenvolvida por Francesco Cirillo. A técnica consiste em dividir o trabalho em intervalos focados de 25 minutos, separados por pequenas pausas, ajudando a manter a concentração e reduzir a fadiga mental.

## ✨ Funcionalidades

- ⏱️ **Timer Pomodoro**: Ciclos de 25 minutos de trabalho focado
- ☕ **Pausas Inteligentes**: Pausas curtas de 5 minutos e longas de 15 minutos
- 📊 **Visualização de Ciclos**: Interface visual mostrando o progresso dos ciclos
- 🎨 **Interface Moderna**: Design dark/light theme com transições suaves
- 📱 **Responsivo**: Funciona perfeitamente em desktop e dispositivos móveis
- 📝 **Gerenciamento de Tarefas**: Adicione e acompanhe suas tarefas
- 📈 **Histórico**: Visualize suas sessões anteriores
- ⚙️ **Configurações Personalizáveis**: Ajuste os tempos conforme sua necessidade

## 🚀 Tecnologias Utilizadas

- **[Next.js](https://nextjs.org/)** - Framework React para produção

## 🛠️ Instalação

### Pré-requisitos

- Node.js 18+ 
- npm, yarn, pnpm ou bun

### Passos

1. Clone o repositório:
```bash
git clone https://github.com/JohnSilva98/chronodoro.git
cd chronodoro
```

2. Instale as dependências:
```bash
npm install
# ou
yarn install
# ou
pnpm install
```

3. Execute o servidor de desenvolvimento:
```bash
npm run dev
# ou
yarn dev
# ou
pnpm dev
```

4. Abra [http://localhost:3000](http://localhost:3000) no seu navegador.

## 📖 Como Usar

1. **Adicione uma tarefa** no campo de texto
2. **Clique em play** para iniciar o timer de 25 minutos
3. **Trabalhe focado** até o timer tocar
4. **Faça uma pausa** de 5 minutos
5. **Repita o ciclo** - após 4 ciclos, faça uma pausa longa de 15 minutos

## 🎮 Navegação

- **Home** (`/`) - Timer principal e gerenciamento de tarefas
- **Histórico** (`/history`) - Visualize suas sessões anteriores
- **Configurações** (`/settings`) - Personalize tempos e preferências
- **Sobre** (`/about`) - Saiba mais sobre a técnica Pomodoro

## 🎨 Temas

O Chronodoro suporta temas claro e escuro. Use o botão de lâmpada no menu para alternar entre os temas.
